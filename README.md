# 📝 To-Do List Python App

A modern GUI-based To-Do List application built using Python and CustomTkinter.
This app helps users manage tasks efficiently with features like priority setting, filtering, analytics, and visualization.

---

## 🚀 Features

✔ Add and delete tasks
✔ Mark tasks as completed
✔ Set task priority (High, Medium, Low)
✔ Add due dates
✔ Filter tasks (Completed, Pending, Priority-wise)
✔ View task analytics
✔ Visualize data using a Pie Chart
